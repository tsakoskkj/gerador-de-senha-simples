# 🔐 Gerador de Senhas Simples

Um script em Python para gerar senhas aleatórias de forma rápida e segura.

## 🚀 Como usar
1. Clone o repositório: `git clone https://github.com/tsakoskkj/gerador-de-senha-simples.git`
2. Entre na pasta: `cd gerador-de-senha-simples`
3. Rode o script: `python3 gerador_senha.py`

## 🛠️ Tecnologias
- Python 3
- Módulos `random` e `string`
